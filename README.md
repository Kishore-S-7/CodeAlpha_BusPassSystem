# 🚌 Smart Cloud Bus Pass System

## 📌 Project Overview

Smart Cloud Bus Pass System is a web-based application developed using Flask, HTML, CSS, and JavaScript. The system allows users to generate a digital bus pass with passenger details, photo verification, QR code verification, and PDF download functionality.

## 🚀 Features

* Passenger Registration
* Photo Upload
* Unique Pass ID Generation
* Verification ID Generation
* Monthly Student Pass
* Issue Date & Expiry Date
* QR Code Verification
* Active Pass Status
* PDF Download Support
* Modern User Interface

## 🛠️ Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* QR Code API

## 📷 Project Workflow

1. User enters passenger details.
2. User uploads a photo.
3. System generates a unique digital bus pass.
4. QR code is generated for verification.
5. Pass can be downloaded as PDF.

## ▶️ How to Run

### Install Dependencies

```bash
pip install flask
```

### Run Application

```bash
python app.py
```

### Open Browser

```text
http://127.0.0.1:5000
```
